# Urban Threads - Modern Clothing E-commerce Website

Urban Threads is a modern, responsive e-commerce website for fashion enthusiasts. Founded in Vadodara in 2024, it offers a curated collection of trendy clothing and accessories, blending timeless silhouettes with contemporary street style.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Product Catalog**: Browse new arrivals, trending items, and curated collections
- **Shopping Cart**: Client-side cart functionality with localStorage persistence
- **Quick View Modals**: Instant product previews without page navigation
- **Collections**: Dedicated pages for Men, Women, Accessories, Summer Vibes, and Modern Workwear
- **User Accounts**: Login, signup, and profile management pages
- **Customer Support**: FAQ, contact, shipping, and size guide pages
- **Newsletter Signup**: Stay updated with latest drops and offers
- **Smooth Animations**: Hover effects and transitions for enhanced user experience

## Technologies Used

- **HTML5**: Semantic markup for structure
- **Tailwind CSS**: Utility-first CSS framework for styling
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Feather Icons**: Lightweight, scalable icons
- **Google Fonts (Inter)**: Modern typography
- **LocalStorage API**: Client-side data persistence for cart

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/urban-threads.git
   cd urban-threads
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - No server setup required for static site


## Usage

1. **Browse Products**: Navigate through different sections using the header menu
2. **View Details**: Click "Quick View" on product cards for instant previews
3. **Add to Cart**: Use the "Add to Cart" button on products
4. **Manage Cart**: Click the cart icon to view, update quantities, or remove items
5. **Explore Collections**: Visit dedicated pages for specific categories
6. **Contact Support**: Use the footer links for customer service
